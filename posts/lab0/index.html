3Demos is a visualization tool developed for presentations in this class. Here, we use it to explore some basic properties of points, vectors, and vector operations.

Goals

  • Visualize various linear combinations of vectors.
  • Explore and utilize some dynamic features.
  • Save a scene in json format to be used later.

Linear Combinations

  1. Navigate to https://3demos.ctl.columbia.edu/.
  2. Click on Objects → Add Object → Vector
  3. Repeat once.

Our scene now has 2 vectors with random components. We will refer to them as $\vec v$ and $\vec w$, respectively. Use the same procedure to add more vectors, and then directly manipulate the vectors’ components to plot the following vector expressions.

  • $ \vec v + \vec w $
  • $ \mathbf v - \mathbf w $ (Plot 2 copies of this, one at the origin and one at $\vec w$.)
  • $ \frac{\mathbf v}{|\mathbf v|} $
  • $ \frac12\mathbf v + \frac12\mathbf w $
  • $ \frac14\mathbf v + \frac34\mathbf w $
  • $\operatorname{proj}_{\mathbf v}\mathbf w $

Dynamic objects

Some objects in 3Demos can be manipulated in time by adding a t variable to the parameter expressions. Try this by appending a * t expressions to each component of one of your vectors. Warning: the object may disappear as t defaults to a value of 0. Use the slider to adjust this value and see the result of scaling your vector. Use the play controls to automate this. Then adjust the range on t to extend the amount of scaling (including negative values).

Construct a new vector with the (convex combination) $$(1 - t) \vec v + t\vec w.$$

Note you must use the * operator for multiplication. The parser may not interpret 5(4) as 20, for example.

Families of vectors

As t-dependence can get us dynamic content, we can also visualize collections of vectors via a discrete parameter n. For instance, see this example of the same vector plotted at multiple positions on the $x$-axis. (We simply set the parameter $p_1 =$ n / 4 and set the range on n from $-4$ to $4$.)

Unit vectors

This works in the vector components as well. For instance, suppose we want 10 evenly spaced unit vectors plotted at the origin in the $xy$-plane.

Theorem Every unit vector in $\R^2$ has the form $\langle \cos \theta, \sin \theta \rangle$ for some angle $\theta$.

(I’ll leave the reader to consider how this is true.) Thus, by dividing the interval $[0,2\pi)$, evenly, we can get the family of unit vectors mentioned above by using the components $x = $ cos(n * pi / 5), $y = $ sin(n * pi / 5), $z = $ 0.

Note you can spell out the word pi for $\pi$.
